# BuildMyBridge - 项目简介

## 🎯 项目目标

BuildMyBridge 是一个开源的飞书机器人集成平台，旨在帮助用户快速集成多种 AI 应用（如 Dify）到飞书消息系统中。

通过 BuildMyBridge，用户可以：
- 在飞书中与 AI 应用进行实时对话
- 管理多个机器人账户和 AI 应用配置
- 追踪消息交互历史
- 灵活配置消息路由规则

## 🛠️ 快速开始

### 前置要求
- Java 17+
- Maven 3.8+
- MySQL 8.0+
- Redis 7.x+
- Node.js 16+

### 5分钟快速启动
```bash
# 1. 配置环境变量
export FEISHU_APP_ID="your-app-id"
export FEISHU_APP_SECRET="your-app-secret"
export JWT_SECRET="your-secret-key"

# 2. 启动后端
cd backend
mvn clean spring-boot:run

# 3. 启动前端（新终端）
cd frontend
npm install
npm run dev

# 4. 访问应用
# 前端: http://localhost:5173
# API文档: http://localhost:8081/api/doc.html
```

详细指南请参考 `docs/01-快速启动.md`

## 🤝 贡献指南

### OpenSpec 工作流

本项目使用 **OpenSpec** - 一个规范驱动的开发工作流：

1. **探索** (`/opsx:explore`) - 思考和调查
2. **提案** (`/opsx:propose`) - 创建变更和设计
3. **实施** (`/opsx:apply`) - 实现代码
4. **归档** (`/opsx:archive`) - 完成变更

所有进行中的工作都跟踪在 `openspec/changes/build-my-bridge/` 中。
