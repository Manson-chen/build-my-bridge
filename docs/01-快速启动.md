# 快速启动指南

## 📋 前置要求检查

在启动应用前，确保已安装以下软件：

### 必需
- ✅ **Java 17 或更高版本**
  ```bash
  java -version  # 应该显示 17+
  ```

- ✅ **Maven 3.8 或更高版本**
  ```bash
  mvn -version   # 应该显示 3.8+
  ```
  
  如未安装，请参考 [Maven 安装指南](./指南/Maven安装指南.md)

- ✅ **MySQL 8.0+**
  ```bash
  # 确保已启动
  mysql --version
  ```

- ✅ **Redis 7.x+**
  ```bash
  # 确保已启动
  redis-cli ping  # 应返回 PONG
  ```

### 可选
- Node.js 16+ （如需启动前端）
- Docker （如需容器化部署）

## 🚀 五分钟快速启动

### Step 1: 配置环境变量

飞书应用信息可从 [飞书开放平台](https://open.feishu.cn) 获取。

#### Linux / macOS
```bash
export FEISHU_APP_ID="cli_your_app_id_here"
export FEISHU_APP_SECRET="your_app_secret_here"
export JWT_SECRET="your-jwt-secret-key-change-this-in-production"
```

#### Windows PowerShell
```powershell
$env:FEISHU_APP_ID="cli_your_app_id_here"
$env:FEISHU_APP_SECRET="your_app_secret_here"
$env:JWT_SECRET="your-jwt-secret-key-change-this-in-production"
```

### Step 2: 启动后端应用

```bash
cd backend
mvn clean compile
mvn spring-boot:run
```

首次运行会自动下载依赖，请耐心等待。

**成功标志**：终端显示 `Started BuildMyBridgeApplication`

### Step 3: 验证后端正常运行

```bash
# 检查健康状态
curl http://localhost:8081/api/health

# 应返回:
# {"status":"UP"}
```

### Step 4: 访问 API 文档

打开浏览器访问：**http://localhost:8081/api/doc.html**

你将看到完整的 API 文档，包括所有端点、请求格式和响应格式。

### Step 5（可选）: 启动前端应用

在新的终端中：

```bash
cd frontend
npm install
npm run dev
```

前端应用将运行在 http://localhost:5173

## 🔧 配置说明

### application.yml

应用配置文件位于 `backend/src/main/resources/application.yml`

关键配置项：

```yaml
# 飞书应用信息
feishu:
  app-id: ${FEISHU_APP_ID:}
  app-secret: ${FEISHU_APP_SECRET:}
  redirect-uri: http://localhost:8080/api/auth/callback

# JWT 配置
jwt:
  secret: ${JWT_SECRET:your-secret-key}
  expiration: 86400000  # 24 小时（毫秒）

# 数据库配置
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/buildmybridge?useSSL=false&serverTimezone=UTC
    username: root
    password: root
    driver-class-name: com.mysql.cj.jdbc.Driver

  redis:
    host: localhost
    port: 6379
    timeout: 2000ms

# 日志
logging:
  level:
    root: INFO
    com.buildmybridge: DEBUG
```

### 环境变量优先级

1. **操作系统环境变量** （最高优先级）
2. **application.yml** 中的值
3. **硬编码默认值** （最低优先级）

例如：
```yaml
feishu:
  app-id: ${FEISHU_APP_ID:default-value}
```

这表示：先从环境变量 `FEISHU_APP_ID` 获取，如果不存在则使用 `default-value`。

## 🐛 常见问题排查

### ❌ 编译失败：找不到飞书 SDK

**错误信息**：
```
[ERROR] Failed to execute goal on project buildmybridge-backend: 
Could not find artifact com.larksuite.oapi:oapi-sdk:jar:2.5.3
```

**解决方案**：
```bash
# 强制更新依赖
mvn clean dependency:resolve --update-snapshots

# 清理缓存后重试
rm -rf ~/.m2/repository/com/larksuite
mvn clean install
```

### ❌ 启动失败：端口被占用

**错误信息**：
```
Address already in use: bind
```

**解决方案**：
```bash
# 方案 1: 改变端口
# 编辑 application.yml:
server:
  port: 8082  # 改为其他端口

# 方案 2: 杀死占用进程 (Linux/Mac)
lsof -i :8081 | grep LISTEN | awk '{print $2}' | xargs kill -9

# 方案 2: 杀死占用进程 (Windows PowerShell)
Get-NetTCPConnection -LocalPort 8081 | Stop-Process -Force
```

### ❌ 认证失败：JWT Token 过期

**错误信息**：
```
401 Unauthorized
```

**解决方案**：
```bash
# 重新执行 OAuth 流程获取新 token
curl -X GET http://localhost:8081/api/auth/login
```

### ❌ 数据库连接失败

**错误信息**：
```
Access denied for user 'root'@'localhost'
```

**解决方案**：
1. 确保 MySQL 已启动
2. 检查数据库用户和密码
3. 创建数据库：
   ```bash
   mysql -u root -p
   CREATE DATABASE buildmybridge;
   ```

### ❌ Redis 连接失败

**错误信息**：
```
Unable to connect to Redis Server
```

**解决方案**：
```bash
# 启动 Redis
# Windows: redis-server
# Linux: redis-server
# Docker: docker run -d -p 6379:6379 redis:latest

# 验证连接
redis-cli ping
# 应返回: PONG
```

## 📊 查看日志

### 启用详细日志

在 `application.yml` 中修改日志级别：

```yaml
logging:
  level:
    root: INFO
    com.buildmybridge: DEBUG
    org.springframework.security: DEBUG
    org.springframework.web: DEBUG
```

### 查看特定模块日志

```bash
# 飞书相关日志
mvn spring-boot:run | grep -i feishu

# 认证相关日志
mvn spring-boot:run | grep -i auth

# 保存完整日志到文件
mvn spring-boot:run > app.log 2>&1
tail -f app.log
```
